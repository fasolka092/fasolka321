Nowe możliwości CSS w porównaniu z HTML

  1. Przestarzałe znaczniki HTML są zastępowane deklaracjami CSS.
  2. Dokumenty są przejrzystsze i krótsze.
  3. Możliwość zarządzania wieloma dokumentami przez zewnętrzne arkusze stylów.
  4. Oszczędność kodu dzięki klasom selektorów.
  5. Szerokie możliwości definiowania kolorów i jednostek miar.
  6. Dodatkowe możliwości formatowania tekstu.
  7. Efektowne filtry graficzne.
  8. Możliwość definiowania wyglądu dokumentów w wersji do druku.

Wady CSS

  • Złożona specyfikacja składni.
  • Brak sygnalizacji błędów.
  • Błędna interpretacja stylów przez różne przeglądarki.
