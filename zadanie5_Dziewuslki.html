Zadanie 5 - Demonstracja sposobów podłączenia CSS

Import

<style>
    @import url("style.css");
</style>

Zewnętrzny arkusz

<link rel="stylesheet" type="text/css" href="style.css">

Wewnętrzny arkusz

<style>
    p { color: blue; font-size: 20px; }
</style>

Liniowy styl

<p style="color: red; font-size: 18px;">Przykładowy tekst</p>
